# BitGainz-Bittrex
A monitoring tool written in [Go](https://golang.org/) and [Bittrex API Go Wrapper](https://github.com/toorop/go-bittrex), to ensure profit and minimum loss when investing in different cryptocurrency markets.
## Requirements
* [Golang 1.9.2](https://golang.org/dl/)
* [MySQL](https://www.mysql.com/)
